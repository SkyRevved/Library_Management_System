# Library Management System

This project is a Library Management System designed to help manage and maintain library records. It consists of a front-end application and a back-end database setup.

## Project Structure

The ZIP file contains the following folders:

1. **library** - Contains the main application files.
2. **SQL** - Contains the SQL database setup file.

## Installation and Setup

### 1. Setup Application

1. **Download and Unzip**: Download the ZIP file and unzip it on your local system.
2. **Move Folder**: Copy the `library` folder into your web server's root directory (e.g., `htdocs` for XAMPP or `www` for WAMP).

### 2. Configure Database

1. **Access phpMyAdmin**: Open phpMyAdmin in your web browser.
2. **Create Database**: Create a new database named `library`.
3. **Import SQL File**: Import the `library.sql` file found in the `SQL` folder into the `library` database.
